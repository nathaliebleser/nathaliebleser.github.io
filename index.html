Theses

Master Thesis Investigation of Intrusion Detection Systems in a Cyber-Physical Laboratory Environment with Respect to Detection Quality

I built a simulated Smart Grid Evironment (a digital twin) of a real-world Smart Grid Laboratory using Docker Containers and Mininet network emulation. Then I developed attacks that could be executed on both environments and collected data during their runs. That data was then used to evaluate how well the attacks were detected by chosen intrusion detection systems on each environment.
